CREATE — DDL оператор языка SQL, используемый для создания объектов базы данных.

INSERT — оператор который вставляет новые строки в существующую таблицу.

SELECT — оператор который применяется для извлечения строк, выбранных из одной или нескольких таблиц.

UPDATE — оператор который обновляет значения существующих столбцов таблицы в соответствии с введенными значениями.

DROP — оператор который является универсальным оператором удаления объектов базы данных. Используя его можно удалить
таблицы (TABLE), индексы (INDEX) и базы данных (DATABASE).
